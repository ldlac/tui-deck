## Features

- **ASCII art** diagrams supported
- **Syntax highlighting** for code blocks
- **Presenter notes** in dual-window mode

## Navigation

- Arrow keys: Navigate slides
- `q`: Quit
- `j`/`k` or `Space`: Next/Previous
